Detailed Description

Shared interrupt 9 routine.

Activate the interrupt by setting EXM = 1.

This interrupt has the following sources:

  • EXINT3/T2CC0
  • EXINT4/T2CC1
  • EXINT5/T2CC2
  • EXINT6/T2CC3
  • CANSRC2
