Detailed Description

Shared non-maskable interrupt routine.

This interrupt has the following sources:

  • Watchdog Timer NMI (NMIWDT)
  • PLL NMI (NMIPLL)
  • Flash Timer NMI (NMIFLASH)
  • VDDP Prewarning NMI (NMIVDDP)
  • Flash ECC NMI (NMIECC)
